Accessibility & Contrast Ratios

WCAG Compliance

All text colors meet WCAG accessibility standards:

  • Seasalt (#f4f7f5) on Void Black: ~14:1 (AAA)
  • Pure White (#fefefe) on Void Black: ~16:1 (AAA)
  • Light Gray (#b8bbbf) on Void Black: ~8:1 (AAA)
  • Dim Gray (#6a6e73) on Void Black: ~5:1 (AA)
  • Bright Teal (#7db5b5) on Void Black: ~8:1 (AAA)
