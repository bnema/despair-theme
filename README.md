# Despair Theme

A dark theme for those who've given up on pastels.

![Despair Color Palette](palette.svg)

## Color Palette

### Background Colors
- `#08090a` Void Black - Primary background
- `#0f1011` Shadow Gray - Borders, subtle dividers
- `#181a1b` Deep Charcoal - Floating panels, input fields
- `#222823` Eerie Black - Active tabs, highlighted sections
- `#2d3236` Slate Gray - Hover states, card backgrounds

### Text Colors
- `#f4f7f5` Seasalt - Main body text (WCAG AAA)
- `#fefefe` Pure White - Emphasized text, selected items
- `#b8bbbf` Light Gray - Secondary information, labels
- `#8b8f94` Silver Gray - Tertiary text, inactive elements
- `#6a6e73` Dim Gray - Comments, hints, placeholders

### Accent Colors - Teal Spectrum
- `#7db5b5` Bright Teal - Keywords, links, active indicators
- `#5a8585` Slate Teal - Types, classes, buttons
- `#3d6666` Deep Teal - Numbers, constants
- `#a7a2a9` Rose Quartz - Muted accents, selected text

### Status Colors
- `#7db5b5` Success Teal - Git additions, success states
- `#d96c75` Soft Red - Errors, deletions, destructive actions
- `#d9a66c` Muted Orange - Warnings, modified states
- `#6c9bd9` Info Blue - Information states, hints

## Supported Apps

### Editors & IDEs
- **Neovim** - `neovim/despair.lua` → `~/.config/nvim/colors/`
- **VSCode** - Install from `vscode/` directory

### Terminal Emulators
- **Alacritty** - `alacritty/despair.toml` → `~/.config/alacritty/themes/`
- **Foot** - `foot/despair.ini` → `~/.config/foot/themes/`
- **Ghostty** - Add config from `ghostty/despair` to your main config
- **Kitty** - Add config from `kitty/despair.conf` to your kitty.conf

### Wayland & Window Managers
- **Hyprland** - `hyprland/despair.conf` → `~/.config/hypr/themes/`
- **Hyprlock** - `hyprlock/despair.conf` → `~/.config/hypr/`
- **Waybar** - Use `waybar/despair.css` in your config
- **Mako** - `mako/despair.ini` → `~/.config/mako/`
- **SwayOSD** - Use `swayosd/despair.css` in your config

### Utilities & Tools
- **Btop** - `btop/despair.theme` → `~/.config/btop/themes/`
- **Eza** - `eza/despair.yml` → `~/.config/eza/`
- **Fuzzel** - `fuzzel/despair.ini` → `~/.config/fuzzel/`
- **Walker** - Use `walker/despair.css` in your config
- **Zellij** - `zellij/despair.kdl` → `~/.config/zellij/themes/`

### Other
- **Chromium** - Icon/theme reference in `chromium/`
- **Icons** - Theme reference in `icons/`
